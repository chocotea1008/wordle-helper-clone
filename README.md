# 꼬들(Wordle) 헬퍼 클론 (100% Frontend 버젼)

이 프로젝트는 한국어 워들(꼬들) 및 각종 변형 게임(5자모, 6자모, 7자모)을 위한 고급 단어 추천 시스템입니다. 

원본 백엔드의 섀넌 엔트로피(Shannon Entropy) 및 기댓값 최적화 알고리즘을 **비트 마스킹(Bitmasking)** 및 메모리 최적화를 통해 100% 프론트엔드 단일 파일 구조(`solver.js`)로 완벽하게 포팅했습니다. 
이를 통해 서버 없이도 브라우저만으로 초고속, 고품질의 추천을 제공합니다.

## ✨ 주요 기능
- **5~7자모 완벽 지원**: 꼬들(5자모), 꼬들6(6자모) 등 다양한 꼬들 변형에 대응합니다.
- **오프라인 100% 프론트엔드**: 서버 API 요청 없이 모든 수학적 연산을 브라우저 내부에서 수행합니다. (평생 무료, 서버 유지비용 0)
- **5가지 고급 추천 알고리즘**:
  1. **균형 추천 (최적)**: 정보량(Entropy)을 계산해 평균적으로 남은 후보를 가장 많이 줄이는 최고의 단어를 추천합니다.
  2. **직접 타격 (정답 후보)**: 남은 정답 후보군 중 가장 유력한 단어를 추천합니다.
  3. **필터링 (초록 배제)**: 확정된 초록색 글자를 의도적으로 제외하여, 완전히 새로운 글자를 최대한 많이 찔러보는 필터링 단어를 추천합니다.
  4. **하드 모드 최적**: 이전 힌트를 전부 준수하면서 후보군을 가장 많이 줄일 수 있는 단어를 추천합니다.
  5. **자모 다양성 (새 자모)**: 아직 테스트하지 않은 새로운 자모들을 가장 많이 포함한 단어를 추천합니다.

## 🚀 알고리즘 최적화 (Bitmasking)
기존 백엔드 API에서 Python으로 수행되던 "2만 4천 단어 전체 스캔" 연산은 브라우저에서 실행하기에 무리가 있었습니다. (약 수천만 번의 문자열 매칭 발생)

이 프로젝트는 이를 해결하기 위해:
1. `getPatternFast`를 도입하여 자모 일치 여부를 **Base-3 숫자**와 **Bitmask** 연산으로 전환했습니다.
2. `Int32Array`를 사용해 메모리 할당 병목을 없애고 C언어에 준하는 속도로 최적화했습니다.
결과적으로 수천만 번의 연산이 브라우저에서도 단시간 내에 완료됩니다.

## 🛠 실행 방법
별도의 빌드나 서버 구축이 필요하지 않습니다.

1. 이 저장소를 다운로드 받거나 `git clone` 합니다.
2. `index.html` 파일을 크롬 등 최신 웹 브라우저로 엽니다.
3. 화면의 안내에 따라 힌트를 입력하고 최적의 단어를 추천받으세요!

## 📁 파일 구조
- `index.html` : UI 및 뷰포트 처리, 모드 전환 등 메인 인터페이스
- `solver.js` : 24,000+ 한국어 단어 사전 및 고급 비트마스크 엔트로피 연산 엔진
- `scripts/` : 사전 데이터를 생성하거나 병합할 때 사용했던 유틸리티 Python/JS 스크립트 모음
- `data/` : 원시 단어 사전 텍스트 데이터
